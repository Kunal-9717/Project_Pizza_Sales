# Project_Pizza_Sales

1. Project Title : "Pizza Sales"

2. Purpose : To understand sales and Revenue by days, months, orders and so on.

3. Tech Stack:
   I made this project using power bi only. I have used following functions:
   - (A) Power BI desktop:- Main data visulation tool used for dashboard creation.
   -(B) Power Query:- Data transfomation and cleaning layer for reshaping tha data.
   -(C) DAX (Data Analysis Expressions):- Used for calculated measures, Dynamic visuals.
   -(D) File Format:- .pbix for development and .png for dashboard previews.

4. Features / Highlights:
  - (A) The business is running good but some products are still make losses.
  - (B) I built the dashboard to identify those pizzas.
  - (C) 'The Brie Carre Pizza' is the one of those pizzas which are less profitable to us.
  - (D) Best Sallers:-
       REVENUE: 'The Thai Chicken pizza' contributes to maximum revenue.
       QUANTITY: 'The Classic Deluxe Pizza' contributes to maximum Total Quantities.
       TOTAL ORDERS: 'The Classic Deluxe Pizza' contributes to maximum Total Orders.

 5. Key insights:
   - (A) We have to stop the production of 'The Brie Carre Pizza'.
   - (B) We should have enough active employees in July & January.
   - (C) Stop making 'xxl' pizzas, because people rarely buy it.
